# 🚀 Geometric Linear Optimization in 4D

## 📌 Overview
This project extends the classical graphical method of linear programming beyond 3 dimensions using a novel **Binary Solution Space approach**.

## 🧠 Key Idea
- Traditional geometric methods work only for 2–3 variables
- This project enables solving **4-variable linear optimization problems**
- Uses:
  - Binary solution space
  - Hyper-tetrahedron intersection
  - Constraint elimination

## 📊 Example Problem

Maximize:
S = 20W + 18X + 3Y + 10Z

Subject to:
15W + 15X + 3.75Y + 3Z ≤ 30  
5W + 8X + 4Y + 5Z ≤ 40  
6W + X + 2Y + 5Z ≤ 10

- Proposed Method: **46.6**
